Navigation

  • At the top of each page is the Top-Navigation-Bar (Inter-Page). This is where you will traverse through every page. Simply select any of the options to navigate to another page.
  • At the bottom of each page is a "End of Page" footer. This contains an ID that the Intra-Page-Navigation-Bar used to know where the bottom of the page is.
  • At the bottom of your screen, following the view as you scroll, is the Bottom-Navigation-Bar (Intra-Page). This is used to ease the use traversal within a webpage. The options on this nav bar will differ from page to page.

 

Go ahead and try out the nav-bars! Jump from page to page, or from section to section within a single page.
